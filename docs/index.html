CryptoBar

Real-time cryptocurrency prices in your macOS menu bar. Lightweight, fast, and beautiful.

Download GitHub

Features

Real-time Prices

Live cryptocurrency prices via WebSocket streaming. Updates instantly without polling.

Multiple Exchanges

Supports Binance, HTX (Huobi), and Gate.io. Switch between them with one click.

1900+ Coins

Search and select from over 1900 cryptocurrencies with a native search window.

Coin Logos

Display beautiful coin icons in your menu bar. Supports color and grayscale modes.

Price Alerts

Set price alerts and get native macOS notifications when targets are hit.

5 Languages

English, 简体中文, 繁體中文, 日本語, 한국어. Full internationalization support.

Native macOS

Built with Go + Objective-C/Cocoa. Native settings window, no Electron bloat.

Lightweight

No Dock icon. Runs quietly in your menu bar. Under 6MB download size.

Installation

Homebrew

brew tap colin-nian/tap
brew install cryptobar

Manual Download

Download the latest .zip from GitHub Releases, unzip and drag to Applications.

Supported Exchanges

Real-time data via WebSocket from major exchanges
